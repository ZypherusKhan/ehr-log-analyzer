### Installation & Development

```bash
# Clone the repository
git clone https://github.com/ZypherusKhan/ehr-log-analyzer.git
cd ehr-log-analyzer

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser

### Build for Production

```bash
# Build the site
npm run build

# Preview production build locally
npm run preview
```

## ğŸ“Š Analyzed Data Types

### Player Sessions
Extract comprehensive player information including names, friend codes, hashed PUIDs, platform identifiers, and client connection metadata.

### RPC Events
Parse all remote procedure calls with per-player event counts, sortable tables, and advanced filtering options for deep analysis.

### Chat Transcript
View complete chat history with accurate timestamps, sender identification, and full-text search capabilities.

### EAC Reports
Monitor Easy Anti-Cheat violations with severity classification (Fatal/Error/Warning) and detailed violation information.
